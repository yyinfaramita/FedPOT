# FedPOT
The code is in the folder named "FedPOT" and mainly contains the files of two attack methods (over-claim and under-train), an incentive mechanism (Refiner), and an aggregation method (FedAvg).
- over-claim.py for over-claim method
- under-train.py for under-train method
- utils: utils.py for aggregation method and local training
- incentive: incentive.py for incentive mechanism
